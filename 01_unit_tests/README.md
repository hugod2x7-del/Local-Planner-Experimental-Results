# Unit Tests

This folder contains unitary validation tests used to evaluate individual components of the local planner on the real RAMBO robotic platform.

These tests were performed before the complete Smart Retail scenarios, with the objective of validating the controller response to individual orientation components and simpler tracking behaviors.

## Purpose

These tests were used to verify whether the local controller was able to follow individual orientation changes before combining multiple motion components in more complex experiments.

They represent the first validation stage of the real robotic implementation.
