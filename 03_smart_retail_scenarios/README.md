# Smart Retail Scenarios

This folder contains supplementary Smart Retail-inspired real-world experiments.

These experiments evaluate the behavior of the local planner in scenarios closer to the intended application, including target acquisition, picking, cluttered environments, and transport.

## Purpose

These tests increase the complexity of the validation process by introducing more realistic manipulation conditions.

They were used to evaluate the behavior of the framework when the target object is placed in environments that resemble retail restocking scenarios.
